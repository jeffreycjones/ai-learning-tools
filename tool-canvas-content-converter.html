Tips for Best Results

Clean Source Content First

Remove obvious page numbers, headers, footers, and references from PDFs before pasting. The cleaner the input, the cleaner the output.

Match Components to Purpose

Use callouts for warnings or key takeaways. Use accordions when content is reference-style. Use tabs when comparing parallel concepts. Do not over-use components.

Always Preview in Canvas

Design Plus styles only render when the page is viewed inside a course where the Design Plus theme is enabled. Preview before publishing to confirm styling.

Verify Accessibility

After pasting into Canvas, run UDOIT (the accessibility checker bundled with Design Plus) to confirm WCAG compliance. The AI gets you 90 percent of the way; UDOIT catches the rest.

Heading Hierarchy Starts at H2

Canvas page titles are H1. Your content headings must start at H2 and not skip levels. The tool follows this rule by default.

Replace Alt Text Placeholders

Where the AI inserts [ALT TEXT NEEDED], replace with a real description before publishing. The Alt Text Generator tool can help.
