Alt Text Best Practices

✓

Convey Meaning, Not Appearance

Focus on what the image conveys to your students, not just describing what you see. Ask: What is the student supposed to learn from this image?

✓

Avoid "Image of..." Phrases

Don't start alt text with "Image of," "Picture of," or "Screenshot of." Assistive technologies already announce that it's an image.

✓

Match Purpose in Context

The alt text should reflect how you use the image in your lesson. The same image might need different alt text in different contexts.

✓

Empty Alt for Decorative Images

Use alt="" for purely decorative images that don't convey information. This prevents screen readers from announcing unnecessary content.

✓

Keep It Concise

Most alt text should be under 125 characters or 1-2 sentences. Longer descriptions work better as captions or in surrounding text.

✓

Describe Data Trends for Charts

For charts and graphs, convey the key data trend or takeaway, not just visual elements. Example: "Revenue growth accelerated 15% year-over-year" rather than "Bar chart with red and blue bars."
